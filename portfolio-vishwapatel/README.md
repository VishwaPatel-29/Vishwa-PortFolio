## 🚀 Quick Start

### 📋 Prerequisites
- ![Node.js](https://img.shields.io/badge/Node.js-18%2B-green?style=for-the-badge&logo=node.js)
- ![npm](https://img.shields.io/badge/npm-8%2B-red?style=for-the-badge&logo=npm)

### ⚡ Installation

1. **🍴 Clone the repository**
   ```bash
   git clone https://github.com/VishwaPatel-29/PortFolio---Vishwa-Patel.git
   cd PortFolio---Vishwa-Patel
   ```

2. **📦 Install dependencies**
   ```bash
   npm install
   ```

3. **⚙️ Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your EmailJS credentials
   ```

4. **🚀 Start the development server**
   ```bash
   npm start
   ```

5. **🌐 Open your browser**
   Navigate to `http://localhost:3000`

## ⚙️ Configuration

### 🌍 Environment Variables

Create a `.env` file in the root directory:

```env
# EmailJS Configuration
REACT_APP_EMAILJS_SERVICE_ID=your_service_id_here
REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id_here
REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key_here
```

### 📧 EmailJS Setup

1. **🔗 Sign up** at [EmailJS](https://www.emailjs.com/)
2. **📧 Add your email service** (Gmail recommended)
3. **📝 Create an email template** with variables:
   - `{{from_name}}`
   - `{{from_email}}`
   - `{{message}}`
4. **📋 Copy your credentials** to the `.env` file

## 🎨 Customization

### 🎨 Theme Customization

Edit `tailwind.config.js` to customize colors and design:

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#your-color',
        secondary: '#your-color',
      }
    }
  }
}
```

### 🖼️ Adding Your Content

1. **📝 Update personal info** in component files
2. **🖼️ Replace images** in the `public/` folder
3. **📧 Update email** in `emailService.js`
4. **🔗 Update social links** in relevant components

## 🚀 Deployment

### 🌐 Netlify
```bash
npm run build
# Deploy the build/ folder to Netlify
```

### 🐳 Vercel
```bash

vercel
```

### 🔧 GitHub Pages
```bash

# Add to package.json:
# "homepage": "https://username.github.io/repo-name"
# "deploy": "gh-pages -d build"
npm run deploy
```
