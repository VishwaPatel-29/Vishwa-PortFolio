## 🚀 Getting Started

### 📋 Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VishwaPatel-29/Vishwa-PortFolio.git
   cd Vishwa-PortFolio/portfolio-vishwapatel
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

### 🏗️ Build for Production
```bash
npm run build
```
